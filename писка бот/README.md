# Телеграм-бот "Линейка" (аналог Pipisa Bot)

Бот для чатов, в котором игроки раз в 24 часа могут использовать команду `/dick`, чтобы случайным образом вырастить или уменьшить "пипису". Также есть топы, глобальный рейтинг и статистика.

## Возможности

- **/dick** — раз в 24 часа случайное изменение размера от -5 до +10 см.
- **/top_dick** — топ‑10 игроков текущего чата.
- **/global_top** — глобальный топ‑10 по всем чатам.
- **/stats** — личная статистика игрока.
- **/help** — помощь и список команд.
- **/buy** — заглушка для покупки дополнительных попыток (можно доработать под реальные платежи).
- **Реклама** — бот два раза в день отправляет рекламные сообщения во все чаты, где он добавлен (тексты настраиваются).

Все результаты хранятся в базе SQLite (`data/bot.db`).

## Установка

1. Установите Python 3.10+.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` рядом с `bot.py` и добавьте туда токен бота:

```bash
BOT_TOKEN=ваш_токен_бота
```

4. Запустите бота:

```bash
python bot.py
```

## Перенос старых результатов

Чтобы перенести результаты из старого бота, потребуется выгрузка данных из его базы (CSV/SQL и т.п.). После этого можно написать небольшой скрипт, который загрузит эти данные в `data/bot.db`. Без доступа к внутренней базе старого бота можно перенести только те цифры, которые вы вручную соберёте (например, из топ‑таблиц).
